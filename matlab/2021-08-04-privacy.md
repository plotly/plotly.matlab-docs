## Public Graphs


```{matlab}
data = {...
  struct(...
    'x', [0, 2, 4], ...
    'y', [0, 4, 2], ...
    'type', 'scatter')...
};

plotly(data, struct('world_readable', true));
```

## Private Graphs


```{matlab}
data = {...
  struct(...
    'x', [0, 2, 4], ...
    'y', [0, 4, 2], ...
    'type', 'scatter')...
};

plotly(data, struct('world_readable', false));
```
