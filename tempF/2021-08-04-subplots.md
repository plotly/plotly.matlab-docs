## Upper and Lower Subplots

```{matlab}
subplot(2,1,1);
x = linspace(0,10);

plot(x,y1)

subplot(2,1,2); 
y2 = sin(5*x);
plot(x,y2)

fig2plotly(gcf)
```

## Quadrant of Subplots

Create a figure divided into four subplots. Plot a sine wave in each one and title each subplot. 

```{matlab}
subplot(2,2,1)
x = linspace(0,10);
y1 = sin(x);
plot(x,y1)
title('Subplot 1: sin(x)')

subplot(2,2,2)
y2 = sin(2*x);
plot(x,y2)
title('Subplot 2: sin(2x)')

subplot(2,2,3)
y3 = sin(4*x);
plot(x,y3)
title('Subplot 3: sin(4x)')

subplot(2,2,4)
y4 = sin(8*x);
plot(x,y4)
title('Subplot 4: sin(8x)')

fig2plotly(gcf)
```
